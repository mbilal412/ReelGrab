# ReelGrab

ReelGrab is a video downloader PWA.

It lets you share a video link from apps (Instagram, YouTube, Facebook, TikTok, X/Twitter, etc.) to ReelGrab and start download automatically.

## Live Use (Easy Guide)

Live Link:
https://reel-grab-indol.vercel.app

### On PC

1. Open the live link.
2. Paste the video URL in the input box.
3. Click Download.

### On Mobile (Chrome)

1. Open the live link in Chrome.
2. Tap the 3 dots menu.
3. Tap Add to Home screen.
4. Open ReelGrab app from home screen.
5. In any app video, tap Share and select ReelGrab.
6. Download starts automatically.

## Supported Platforms

ReelGrab works with many major platforms (for example YouTube, Instagram, Facebook, TikTok, X/Twitter) based on supported public links.

## Local Development

### Backend

1. Go to Backend folder.
2. Install dependencies: npm install
3. Add .env values:
   - PORT
   - FRONTEND_URL
4. Start server: npm run dev

### Frontend

1. Go to Frontend folder.
2. Install dependencies: npm install
3. Add .env value:
   - VITE_BACKEND_URL
4. Start app: npm run dev
